# FloraVision

## 🛠️ Tech Stack

- React JS
- Vite
- Tailwind CSS

## 📂 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/flora-vision.git
```

Go to project folder:

```bash
cd flora-vision
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

## 🌐 Deployment

Project deployed on Vercel.
